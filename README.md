# श्री संत कृपा प्रसाद मोटर ड्रायव्हिंग स्कूल - Website

A modern, animated website for श्री संत कृपा प्रसाद मोटर ड्रायव्हिंग स्कूल, Savda. Built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Modern Design**: Clean, professional UI with extensive animations
- **Driving-themed Animations**: Car movements, steering wheel rotations, floating elements
- **Responsive**: Mobile-first design that works on all devices
- **Multi-language**: Support for Marathi and English text
- **Interactive Elements**: Smooth scrolling, hover effects, animated counters
- **Contact Form**: Full-stack form handling with backend API
- **Service Showcase**: Comprehensive listing of all driving school services
- **Insurance Partners**: Display of 21+ insurance companies
- **Performance Optimized**: Fast loading with optimized assets

## Tech Stack

### Frontend
- React 18 with TypeScript
- Tailwind CSS for styling
- Framer Motion for animations
- Shadcn/ui component library
- React Hook Form for form handling
- Lucide React for icons
- Wouter for routing

### Backend
- Express.js server
- TypeScript
- In-memory storage (production ready for database)
- RESTful API endpoints
- Form validation with Zod

### Deployment
- Optimized for Vercel deployment
- Static asset optimization
- Environment variable support
